ubuntu16.04下安装MySql8.0

Jun 22, 2018点击

一、下载，安装

官方下载地址：
https://dev.mysql.com/downloads/mysql/

最后两个选项是已经经过编译的文件。
我们选择64位的文件(mysql-8.0.11-linux-glibc2.12-x86_64.tar)下载。
我们将压缩包下载到 /usr/local/src 目录。
解压文件 tar 文件，将会得到2个 tar.gz 文件

继续解压其中的不带 test 的文件

然后将解压出来的目录，重命名复制到 /usr/local/mysql

然后进入 /usr/local/mysql 进行安装

basedir：安装目录路径
datadir：数据库数据的根路径
initialize：生成随机密码，使用-initialize-insecure生成空密码。默认帐号root。
datadir: 数据库data存储目录

上述命令安装完成后，会有类似于下发的提示，告知你随机生成的 root 账户的密码，如：tHPie-2CQoGP

然后再继续执行以下命令

service mysql.server start #启动mysql

service mysql.server stop #关闭mysql

systemctl enable mysql.server

mysql> show databases;
ERROR 1820 (HY000): You must reset your password using ALTER USER statement before executing this statement.

alter user ‘root‘@’localhost’ IDENTIFIED BY ‘123456’

alter user ‘root‘@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘123456’

/usr/local/mysql/bin/mysql –verbose –help | grep -A 1 ‘Default options’

cd /usr/local/mysql

mkdir etc

vi my.cnf

`
在 my.cnf 文件中，添加如下

[mysqld]
default_authentication_plugin=mysql_native_password

然后重启MySQL
